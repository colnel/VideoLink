# VideoLink
# 开发时间：2025-11-13
## 项目简介
VideoLink是一个基于.NET技术栈开发的视频服务平台，旨在提供高效的视频资源分享与管理解决方案。项目核心模块涵盖了GB28181和SIP协议的实现，支持流媒体转发功能，并配备了一个用户友好的Web应用界面，方便用户进行视频播放和管理操作。
## 主要功能
- **GB28181和SIP协议支持**：实现了对GB28181和SIP协议的完整支持，确保与各种视频设备的兼容性。
- **流媒体转发**：提供高效的流媒体转发功能，支持多种视频格式和传输协议，确保视频播放的流畅性和稳定性。
- **Web应用界面**：开发了一个直观易用的Web应用界面，用户可以方便地浏览、播放和管理视频资源。
- **用户管理**：支持多用户登录和权限管理，确保视频资源的安全性。
- **视频播放**：集成了多种视频播放器，支持在线播放和下载功能。
## 模块组成
- **SIPProtocol**：负责SIP协议的解析与处理。
- **MANSProtocol**：实现MANS协议(GB28181)的相关功能。
- **StreamServer**：处理流媒体的转发和分发。
- **VideoServer**：核心视频服务模块，负责视频的存储和管理。
- **VideoApp**：提供用户交互界面，支持视频播放和管理。
